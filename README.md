## 💊 Controle de Medicamentos

Projeto desenvolvido como exercício da Academia do Programador.

O sistema auxilia no controle de medicamentos, estoque, pacientes, fornecedores e movimentações de entrada e saída.

---

## 📌 Funcionalidades

### 🚚 Módulo de Fornecedores
- Cadastrar fornecedores
- Editar fornecedores
- Excluir fornecedores
- Visualizar fornecedores

### 🧑 Módulo de Pacientes
- Cadastrar pacientes
- Editar pacientes
- Excluir pacientes
- Visualizar pacientes

### 💊 Módulo de Medicamentos
- Cadastrar medicamentos
- Editar medicamentos
- Excluir medicamentos
- Visualizar medicamentos
- Controle de estoque

### 👨‍💼 Módulo de Funcionários
- Cadastrar funcionários
- Editar funcionários
- Excluir funcionários
- Visualizar funcionários

### 📦 Módulo de Estoque
- Registrar entradas de medicamentos
- Registrar saídas de medicamentos
- Atualização automática do estoque
- Controle de disponibilidade dos medicamentos
